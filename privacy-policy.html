Privacy Policy

PairPlate Content Manager — Last updated: April 12, 2026

1. Introduction

PairPlate Content Manager ("the App") is a personal content management tool designed for the creator of the TikTok cooking channel @PairPlate. This privacy policy explains how the App handles data.

2. Data We Collect

The App collects and stores the following data locally on the user's device:

  • TikTok Account Information: Basic profile data (username, display name) obtained through TikTok's Login Kit with the user's explicit consent.
  • Authentication Tokens: OAuth access tokens and refresh tokens required to publish content on behalf of the authenticated TikTok account.
  • Media Files: Photos and videos stored locally that the user chooses to publish to TikTok.

3. How We Use Your Data

Data is used solely to:

  • Authenticate with TikTok's API to enable content publishing.
  • Upload and publish video content to the authenticated TikTok account.
  • Display basic account information within the App interface.

4. Data Storage & Security

All data is stored locally on the user's personal computer. No data is transmitted to any external servers other than TikTok's official API endpoints during authentication and content publishing. No data is stored in the cloud or on remote servers controlled by the App developer.

5. Third-Party Sharing

We do not sell, trade, or share any user data with third parties. Data is only transmitted to TikTok's official API as required to provide the App's functionality (authentication and content publishing).

6. Data Retention & Deletion

Users can delete all stored data at any time by removing the App's local configuration files. Authentication tokens can be revoked through TikTok's account settings. No data persists after deletion.

7. User Rights

As this is a single-user, locally-run application, the user has full control over all data at all times. The user can:

  • View all stored data in the App's local files.
  • Delete any or all data by removing local files.
  • Revoke TikTok access at any time through TikTok's settings.

8. Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected on this page with an updated date.
